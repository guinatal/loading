<h1>Loading</h1>
<p>Loading to your web project. Supported web mobile./p>
<h2>Usage</h2>
<p>Add loading.js to your webpage. * You also need jquery</p>
<h2>Show loading</h2>
    loading.initialize();
<h2>Close loading</h2>
    loading.close();
<h2>Checking loading</h2>
<p>It returns true if the loading is initialized and false if not.</p>
    loading.exists();
